# INCY

Профиль сохраняет выборочную маршрутизацию проекта: только список
`russia-inside`, Telegram, WhatsApp, Discord и точечные Apple CDN идут через
VPN; остальное — напрямую.

## Рекомендуемая установка с автообновлением

После публикации репозитория откройте ссылку из `autorouting.deeplink`. INCY
загрузит `routing.json` из GitHub, активирует профиль и будет периодически
проверять обновления источника.

Для разового импорта можно использовать `routing.deeplink`.

После изменения JSON выполните из корня репозитория:

```sh
./scripts/build-client-profiles.sh
```

Формат профиля и ссылки сверены с официальной документацией INCY по
[маршрутизации](https://docs.incy.cc/en/routing/) и
[автообновлению](https://docs.incy.cc/en/autorouting/).
