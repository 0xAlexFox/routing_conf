# Happ

Профиль сохраняет выборочную маршрутизацию проекта: только список
`russia-inside`, Telegram, WhatsApp, Discord и точечные Apple CDN идут через
VPN; остальное — напрямую.

## Установка

Откройте на устройстве содержимое `routing.deeplink` как ссылку либо
скопируйте ссылку целиком в Happ. Она добавит и сразу активирует профиль.

Исходный человекочитаемый профиль находится в `routing.json`. После изменения
JSON выполните из корня репозитория:

```sh
./scripts/build-client-profiles.sh
```

Happ обновляет подключённый `geosite.dat`, но сам JSON-профиль из отдельного
URL автоматически не перечитывает. После изменения массивов правил следует
повторно открыть обновлённый deeplink.

Формат профиля и схема ссылки сверены с
[официальной документацией Happ](https://www.happ.su/main/dev-docs/routing).
